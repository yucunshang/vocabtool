# Processed Vocabulary Files

Generated by:

- `tools/build_priority_vocab.py`
- `tools/build_local_card_lexicon.py`

## Files

- `ngsl_31k_priority.csv`: app-facing `word,rank` file.
- `ngsl_31k_priority_metadata.csv`: audit file with scores, original ranks,
  curated-list hits, supplementary categories, and source labels.
- `local_card_lexicon.csv`: local card-definition data used before AI card
  fallback. It contains local English definitions plus optional phonetics,
  POS, and examples from NGSL, NAWL, TSL, and Princeton WordNet via the NLTK
  corpus package. Card generation uses local definitions/examples when the
  relevant fields exist; missing fields are filled by AI and marked in the
  card source note.

## Ranking Inputs

The priority score uses:

- original NGSL+SFI 31k rank as the frequency backbone
- NGSL 1.2 core rank
- NGSL supplementary headwords for weekdays, months, and numbers
- Oxford 3000/5000 and CEFR levels
- NAWL, TSL, and BSL special-purpose ranks

The app only needs `word,rank`, but the metadata CSV should be checked whenever
the scoring formula changes.

## Capitalization

Matching is case-insensitive, but display spelling is preserved for proper
calendar words from NGSL supplementary data, such as `Monday` and `January`.
